# Task manager

## Идея

Task manager — это инструмент для исполнения задач. 

## Возможные features

* Отложенное исполнение задач
* Соблюдение зависимостей между задачами
* Параллельная обработка заданных типов задач
* Повышенная надежность через паттерн Transactional Outbox

## Архитектура

* Состоит из 2 микросервисов: planner и worker. 
* Planner предоставляет CLI. 
* Общение между planner и worker происходит через kafka. 
* Каждый из микросервисов работает с SQL базой данных для хранения состояния.
